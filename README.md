# mosstat_dashboard
Проект по сбору, обработке и анализу статистики Москвы с Мосстата. Включает парсер данных, очистку, загрузку в PostgreSQL, REST API и дашборд в Cubisio.

Проект автоматизирует сбор, предобработку и анализ статистических данных Москвы с сайта Мосстата. 

## Особенности проекта
1. **Сбор данных:** парсер скачивает Excel/CSV-файлы с сайта Мосстата по ключевым показателям:
   - Индекс потребительских цен (ИПЦ)
   - Доходы, расходы и сбережения населения
   - Уровень бедности
   - Заболеваемость населения
   - Численность медицинских кадров

2. **Предобработка и анализ:** очистка данных, визуализация динамики показателей с помощью Matplotlib и Seaborn, вывод предположений о трендах.

3. **База данных PostgreSQL:** проектирование таблиц и загрузка обработанных данных.

4. **Сервис для работы с данными:** 
   - Автоматическое скачивание и обработка данных
   - Загрузка в базу
   - REST API для получения данных пользователем

5. **Дашборд в Cubisio:** визуализация ключевых показателей (линейные графики, круговые диаграммы, гистограммы) с возможностью анализа динамики и структуры данных.

## Технологии
- Python (pandas, requests, matplotlib, seaborn, sqlalchemy, fastapi)
- PostgreSQL
- Cubisio (BI-система для визуализации)

## Как использовать
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить скрипт `service.py` для загрузки данных и запуска API
3. Подключить Cubisio к PostgreSQL для визуализации
